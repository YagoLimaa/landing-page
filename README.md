# Vite + React + Shadcn + TypeScript

This is a template for a Vite project with React, Shadcn UI, and TypeScript.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

## Building for Production

To build the project for production, run the following command:

```bash
npm run build
```

## PWA

This project is a Progressive Web App (PWA). To build the PWA and preview it, run the following command:

```bash
npm run pwa
```

To serve the PWA, run the following command:

```bash
npm run start
```
